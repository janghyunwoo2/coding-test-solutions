## 🔍 문제 이해하기

> - 10진수를 받아서 2진수로 바꾸기


## 💡 핵심 아이디어 / 접근 방식

1.  **문제 이해 및 분석**:
    *   입력
        > 10 숫자
    *   출력
        > "1010" 문자열
    *   예외
        > 입력값은 1000만 이하 양의 정수
    * 분석 요약
        ``` 
        말그대로 풀면 될듯
        ```
2.  **알고리즘 설계**:
    *   **아이디어 1**
        ``` 
        10진수를 2진수로 바꿔주는 함수가 있을 것 같음 => bin()
        bin의 시간 복잡도는 logN 이라고 함
        ```
        *   시간 복잡도: `O(logN)` 
        *   공간 복잡도: `O(N)` 
    *   **아이디어 2**
        ``` 
        bin 함수를 내가 구현해보기로 함
        10진수를 2진법으로 바꾸는 공식을 코드로 표현하기
        ```
        *   시간 복잡도: `O(logN)` 
        *   공간 복잡도: `O(N)`
